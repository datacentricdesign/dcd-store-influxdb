InfluxDB store for the Data-Centric Design Hub.

## Setup

Create file .env with the following variables

```text
LOG_LEVEL=DEBUG
INFLUXDB_HOST=localhost
KAFKA=true
KAFKA_HOST=
KAFKA_PORT=9092
```
